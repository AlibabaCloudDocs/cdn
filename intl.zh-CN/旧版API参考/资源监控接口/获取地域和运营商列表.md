# 获取地域和运营商列表

调用DescribeCdnRegionAndIsp获取地域和运营商列表。

## 调试

[您可以在OpenAPI Explorer中直接运行该接口，免去您计算签名的困扰。运行成功后，OpenAPI Explorer可以自动生成SDK代码示例。](https://api.aliyun.com/#product=Cdn&api=DescribeCdnRegionAndIsp&type=RPC&version=2014-11-11)

## 请求参数

|名称|类型|是否必选|示例值|描述|
|--|--|----|---|--|
|Action|String|是|DescribeCdnRegionAndIsp|系统规定参数。取值：**DescribeCdnRegionAndIsp**。 |

## 返回数据

|名称|类型|示例值|描述|
|--|--|---|--|
|Isps|Array of Isp| |运营商列表。 |
|Isp| | | |
|NameEn|String|unicom|运营商英文名称。 |
|NameZh|String|联通|运营商中文名称。 |
|Regions|Array of Region| |地域列表。 |
|Region| | | |
|NameEn|String|liaoning|地域英文名称。 |
|NameZh|String|辽宁省|地域中文名称。 |
|RequestId|String|2387C335-932C-4E1E-862C-1C4363B6DE72|请求ID。 |

http(s)://cdn.aliyuncs.com/?Action=DescribeCdnRegionAndIsp
